05 — Layout e Grid

The Square.

Uso predominante do quadrado e formas retilíneas, sempre com ângulos retos de 90°. Layouts devem parecer feitos por humanos, como uma colagem — casual, limpo, sem esforço aparente.

O quadrado ecoa a solidez e simetria do símbolo Spark — é o elemento que estabelece ritmo e consistência entre layouts e mídias.

Regra dura
Não usar cantos arredondados nem círculos. Usar apenas quadrados e retângulos com ângulos de 90° exatos. Formatos companheiros citados no guideline: 9:16, 16:9 e 4:5.

Sistema "digital collage"

Categorias de composição, usadas como ponto de partida — não como grade obrigatória fixa:

Grid

Organização em grade.

Layer

Sobreposição de elementos.

Path

Composição ao longo de um caminho/trajetória.

Random

Distribuição aparentemente aleatória, mas estruturada.

Singular

Um elemento central dominante.

Abstract graphic

Elemento gráfico abstrato.

Dois usos do digital collage

  • Housing imagery — usar quadrados/formas retilíneas como moldura/container de imagens.
  • Graphic element — usar quadrados/formas retilíneas como elemento gráfico autônomo (sem imagem dentro).

Checklist ao montar um layout customizado

  • Uso proeminente de quadrado e retângulos.
  • Composições simples, em camadas, editoriais.
  • Espaço negativo generoso.
  • Contraste de escala ou repetição controlada de elementos.
  • Estruturado, mas sem alinhamento perfeito/mecânico — o desalinhamento leve é intencional (tom "Human").
Don't
Layouts excessivamente complexos. Outras formas geométricas quando um quadrado/retângulo resolveria. Cantos arredondados — sempre 90°. Sobrecarregar comunicações com excesso de elementos gráficos simultâneos.

Sem capacidade de design custom? O guideline recomenda partir dos templates de apresentação oficiais — ver Aplicações.
