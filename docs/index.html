Brand Identity Guidelines — Versão 1, Janeiro 2026

O manual de marca da VML.

Human. Confident. Premium. Este manual reúne, em um único lugar navegável, as regras normativas do guideline oficial — logo, cor, tipografia, layout, elementos de design e a filosofia Human First — junto com os assets reais para aplicá-las.

Human

Textura, calor, hand-drawn, colagem digital, desalinhamento intencional — sutil, nunca bagunça.

Confident

Presença sem esforço aparente. Discreta. Uma presença quieta, não gritante.

Premium

Minimalismo refinado, cor reduzida e comedida, tipografia quieta.

Regra de ouro
Na dúvida entre "mais" ou "menos" — mais cor, mais elemento gráfico, mais peso tipográfico, mais Spark — o guideline sempre resolve para menos. A marca está migrando de uma fase grande e ousada (2023) para uma fase contida e premium (2026).

Sobre esta pasta

Este manual foi gerado a partir do pacote de conhecimento de marca em ../agente/ — a mesma base, em formato markdown autossuficiente, que agentes de IA usam para gerar e avaliar peças VML. Os dois falam a mesma regra; este manual é a versão navegável para leitura humana.
